University Marketplace Platform

Role: Backend Developer

Team: Collaborative project with university peers

Project Description: Developed a comprehensive platform for university students to buy, rent, and sell products exclusively within university premises. The platform aims to create a secure and convenient marketplace tailored to the needs of students.

Responsibilities:

  • Designed and implemented the backend infrastructure using Firebase for real-time data storage and user authentication.
  • Developed user registration and authentication features using Firebase Authentication.
  • Managed user data and product listings in Firebase Firestore.
  • Collaborated with front-end developers to ensure seamless integration and functionality.

Technologies Used: Firebase Authentication, Firebase Firestore, React, HTML, CSS, JavaScript
